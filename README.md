## Опис проєкту
Застосунок є спрощеною системою обліку особистої бібліотеки. Головна мета роботи — реалізація багатоекранної архітектури з передачею аргументів між екранами та керуванням станом додатку за допомогою архітектурного підходу **MVI/MVVM**.

### Екрани застосунку:
1. **BooksListScreen (Список книг):** - Відображає динамічний список усіх наявних книг (назва + автор + зірковий рейтинг).
   - Містить плаваючу кнопку `+` для переходу на форму додавання нової книги.
   - По кліку на картку книги здійснюється перехід на екран деталей із передачею унікального `bookId`.
2. **AddEditBookScreen (Додавання / Редагування):**
   - Універсальна форма з полями введення: назва, автор та рік видання.
   - Кнопка збереження автоматично валідує поля і блокується, якщо назва чи автор порожні.
   - Залежно від контексту працює у режимі створення нової книги або оновлення полів існуючої.
3. **DetailsBookScreen (Детальна інформація):**
   - Показує повні дані про книгу (включаючи рік видання).
   - Містить інтерактивний `Slider` для виставлення рейтингу від 1 до 5 зірок, що реактивно оновлює інтерфейс.
   - Надає можливість перейти до редагування книги або безповоротно видалити її з бібліотеки.
